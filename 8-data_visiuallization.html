1. Matplotlib Intro

What is Matplotlib?

  • Matplotlib is a comprehensive Python library for creating static, animated, and interactive visualizations.
  • Created by John D. Hunter in 2003.
  • Works well with NumPy and Pandas.
  • Widely used in data science, machine learning, and scientific research.

Key Features

  • Generate plots, histograms, bar charts, scatter plots, etc.
  • Highly customizable (colors, labels, grid, etc.)
  • Supports multiple backends and output formats (PNG, PDF, SVG, etc.)
  • Integration with Jupyter Notebooks and other IDEs

Installation

pip install matplotlib

Import Convention

import matplotlib.pyplot as plt
import numpy as np

2. Matplotlib Get Started

Basic Syntax

import matplotlib.pyplot as plt
import numpy as np

x = np.array([1, 2, 3, 4])
y = np.array([10, 20, 30, 40])

plt.plot(x, y)
plt.show()

Plotting with Pyplot

plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
plt.show()
Key Points:
plt.plot() is used to draw lines and markers.
plt.show() displays the figure.
Without plt.show(), the plot won't appear.

3. Matplotlib Pyplot

What is Pyplot?

  • Pyplot is a collection of functions in Matplotlib that makes plotting work like MATLAB.
  • It provides a MATLAB-like interface.
  • Each function makes some change to a figure (creates figure, plotting area, plots lines, adds labels, etc.)

Common Pyplot Functions

plt.plot(x, y)          # Line plot
plt.scatter(x, y)       # Scatter plot
plt.bar(x, y)           # Bar chart
plt.hist(data)          # Histogram
plt.pie(data)           # Pie chart

# Customization
plt.title("Title")
plt.xlabel("X Label")
plt.ylabel("Y Label")
plt.grid(True)
plt.legend()
plt.show()

Multiple Plots in One

plt.plot(x, y1, label="Line 1")
plt.plot(x, y2, label="Line 2")
plt.legend()
plt.show()

8. Matplotlib Grid

Adding Grid

plt.grid()

Customizing Grid

plt.grid(axis='x')           # Grid only on x-axis
plt.grid(axis='y')           # Grid only on y-axis
plt.grid(color='red', linestyle='--', linewidth=0.5)

Grid Parameters

  • axis: 'both' (default), 'x', or 'y'
  • color: Grid line color
  • linestyle: '-', '--', '-.', ':'
  • linewidth: Thickness of grid lines
  • alpha: Transparency (0 to 1)

Example

plt.plot(x, y)
plt.grid(True, color='gray', linestyle='dotted', alpha=0.5)
plt.show()

Removing Grid

plt.grid(False)

9. Matplotlib Subplot

What are Subplots?

  • Subplots allow you to display multiple plots in one figure.
  • They are arranged in rows and columns.

Basic Syntax

plt.subplot(rows, columns, index)

Example: 2x2 Subplots

plt.subplot(2, 2, 1)
plt.plot(x, y1)
plt.title("Plot 1")

plt.subplot(2, 2, 2)
plt.plot(x, y2)
plt.title("Plot 2")

plt.subplot(2, 2, 3)
plt.plot(x, y3)
plt.title("Plot 3")

plt.subplot(2, 2, 4)
plt.plot(x, y4)
plt.title("Plot 4")

plt.tight_layout()
plt.show()

Alternative Method: subplots()

fig, axes = plt.subplots(2, 2)

axes[0, 0].plot(x, y1)
axes[0, 1].plot(x, y2)
axes[1, 0].plot(x, y3)
axes[1, 1].plot(x, y4)

plt.show()

Customizing Subplots

fig, axes = plt.subplots(2, 2, figsize=(10, 8))
axes[0, 0].set_title("Title 1")
axes[0, 0].set_xlabel("X Label")
axes[0, 0].set_ylabel("Y Label")

12. Matplotlib Histograms

What is a Histogram?

  • A histogram shows the distribution of numerical data.
  • It divides data into bins and counts frequency.

Basic Histogram

data = np.random.randn(1000)
plt.hist(data)
plt.show()

Customizing Histogram

plt.hist(data, 
         bins=30,
         color='skyblue',
         edgecolor='black',
         alpha=0.7,
         density=True)

Multiple Histograms

13. Matplotlib Pie Charts

What is a Pie Chart?

A pie chart shows proportions of a whole as circular slices.

Basic Pie Chart

sizes = [30, 25, 20, 15, 10]
labels = ['A', 'B', 'C', 'D', 'E']

plt.pie(sizes, labels=labels)
plt.show()

Customizing Pie Chart

plt.pie(sizes, 
        labels=labels,
        colors=['red', 'blue', 'green', 'orange', 'purple'],
        startangle=90,
        explode=(0.1, 0, 0, 0, 0),
        autopct='%1.1f%%',
        shadow=True,
        wedgeprops={'edgecolor': 'black'})
plt.show()

Exploding Slices

explode = [0.1, 0, 0.05, 0, 0]
plt.pie(sizes, labels=labels, explode=explode)

Customizing Labels

plt.pie(sizes, 
        labels=labels,
        autopct=lambda p: f'{p:.1f}%',
        textprops={'fontsize': 12, 'color': 'white'})

Donut Chart

plt.pie(sizes, labels=labels, wedgeprops=dict(width=0.3))
plt.show()

14. Reading Data from CSV and Plotting

Why Read from CSV?

  • CSV is the most common format for storing tabular data.
  • Allows you to plot real-world datasets.
  • Works seamlessly with Pandas and NumPy.

Method 1: Python's Built-in csv Module

import csv
import matplotlib.pyplot as plt

with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    header = next(reader)
    x = []; y = []
    for row in reader:
        x.append(float(row[0]))
        y.append(float(row[1]))

plt.plot(x, y)
plt.xlabel(header[0])
plt.ylabel(header[1])
plt.title('Data from CSV')
plt.show()

With Headers using DictReader

with open('data.csv', 'r') as file:
    reader = csv.DictReader(file)
    for row in reader:
        x.append(float(row['Date']))
        y.append(float(row['Value']))

Method 2: Using NumPy

import numpy as np

data = np.genfromtxt('data.csv', delimiter=',', skip_header=1)
x = data[:, 0]
y = data[:, 1]

plt.plot(x, y, 'bo-')
plt.show()

With Column Names

data = np.genfromtxt('data.csv', delimiter=',', names=True)
x = data['Date']
y = data['Value']

Handling Different CSV Formats

Different Delimiters

df = pd.read_csv('data.csv', delimiter=';')
df = pd.read_csv('data.csv', delimiter='|')

Missing Values

df = pd.read_csv('data.csv', na_values=['NA', 'Missing', '-'])
df = df.dropna()
df = df.fillna(0)

Specific Columns

df = pd.read_csv('data.csv', usecols=['Date', 'Value'])
df = pd.read_csv('data.csv', names=['Date', 'Value'], skiprows=1)

Date Parsing

df = pd.read_csv('data.csv', parse_dates=['Date'])

Data Cleaning and Preparation

df = pd.read_csv('data.csv')

print(df.info())
print(df.describe())

df = df.drop_duplicates()
df = df.dropna(subset=['Value'])
df['Date'] = pd.to_datetime(df['Date'])

df['Month'] = df['Date'].dt.month
df['Year'] = df['Date'].dt.year
df['DayOfWeek'] = df['Date'].dt.day_name()

monthly_avg = df.groupby('Month')['Value'].mean()

monthly_avg.plot(kind='bar')

Complete Example: End-to-End CSV to Plot

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np

df = pd.read_csv('sales_data.csv')

print("Data Info:")
print(df.info())
print("\nFirst 5 rows:")
print(df.head())

df['Date'] = pd.to_datetime(df['Date'])
df['Revenue'] = df['Quantity'] * df['Price']
df['Month'] = df['Date'].dt.month_name()

fig = plt.figure(figsize=(15, 10))

ax1 = plt.subplot(2, 2, 1)
monthly_revenue = df.groupby('Date')['Revenue'].sum()
ax1.plot(monthly_revenue.index, monthly_revenue.values, 'b-o')
ax1.set_title('Daily Revenue Trend')
ax1.grid(True)

ax2 = plt.subplot(2, 2, 2)
product_revenue = df.groupby('Product')['Revenue'].sum().sort_values(ascending=False)
product_revenue.plot(kind='bar', ax=ax2)
ax2.set_title('Revenue by Product')

ax3 = plt.subplot(2, 2, 3)
top_products = df.groupby('Product')['Quantity'].sum().nlargest(5)
ax3.pie(top_products.values, labels=top_products.index, autopct='%1.1f%%')
ax3.set_title('Top Products Distribution')

ax4 = plt.subplot(2, 2, 4)
ax4.scatter(df['Price'], df['Quantity'], alpha=0.5, c='green')
ax4.set_title('Price vs Quantity')

plt.tight_layout()
plt.savefig('sales_analysis.png', dpi=300, bbox_inches='tight')
plt.show()

print("\nSummary Statistics:")
print(df[['Revenue', 'Quantity', 'Price']].describe())

df.to_csv('cleaned_sales_data.csv', index=False)
