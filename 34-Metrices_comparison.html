📊 Regression Metrics R² · MAE · MSE · RMSE · Adjusted R²

Common evaluation metrics used in regression models — explained with examples and simple language.
Metric Full Name Meaning Better Value
R² R-Squared (Coefficient of Determination) Measures how well the model explains the variation in the data. Closer to 1
MAE Mean Absolute Error Average of the absolute prediction errors. Closer to 0
MSE Mean Squared Error Average of the squared prediction errors. Penalizes large errors more heavily. Closer to 0
RMSE Root Mean Squared Error Square root of MSE. Error is in the same unit as the target variable. Closer to 0
Adjusted R² Adjusted R-Squared Similar to R² but penalizes adding unnecessary features. Closer to 1

1. R² (R-Squared) Goodness of fit

Measures how much of the variation in the target variable is explained by the model. Range: 0 to 1 (can be negative for very poor models).

R² = 0.95 → Model explains 95% of the data variation. R² = 0.60 → Model explains 60% of the variation.
Interpretation
1.0 = Perfect prediction  ·  0.8–1.0 = Excellent  ·  0.6–0.8 = Good  ·  <0.5 = Weak model
🧠 Simple language: “My model explains this much of the data.”
R² = 0.90 → Model understands 90% of the data.

2. MAE (Mean Absolute Error) Average error

Average of the absolute differences between actual and predicted values. Treats all errors equally.

MAE = (1/n) · Σ |y − ŷ|
Example:
Actual: 100, 120, 150  ·  Predicted: 95, 125, 145  ·  Errors: 5, 5, 5
MAE = (5+5+5) / 3 = 5
Interpretation: On average, predictions are 5 units away from the true values.
🗣️ Simple: “On average, how many units am I wrong?”  —  Lower is better.

3. MSE (Mean Squared Error) Penalizes large errors

Squares each error before averaging. Large errors have a much bigger impact.

MSE = (1/n) · Σ (y − ŷ)²
Example: Errors: 5, 5, 10  →  Squared: 25, 25, 100
MSE = (25+25+100) / 3 = 50
Use when: You want to strongly penalize large prediction mistakes.
🗣️ Simple: “How bad are my mistakes, especially big ones?”  —  Lower is better.

4. RMSE (Root Mean Squared Error) Same unit as target

Square root of MSE. Easier to understand because it's in the same units as the target.

RMSE = √MSE
Example: MSE = 50  →  RMSE = √50 ≈ 7.07
🗣️ Simple: “Average mistake in the original units.”  —  Lower is better.

5. Adjusted R² Penalizes extra features

Unlike R², Adjusted R² decreases if you add unnecessary features. R² always stays the same or increases when new features are added.
Adjusted R² only increases if the new feature genuinely improves the model.

✅ Better metric for multiple linear regression — helps identify whether adding more features is really beneficial.
🗣️ Simple: “Is adding more features actually helping?”  —  Higher is better.

🧭 Which metric should you use?

  • ✅ R² → How well the model fits the data.
  • ✅ MAE → Average prediction error (easy to interpret).
  • ✅ MSE → Penalize large errors more heavily.
  • ✅ RMSE → Error in the original units (most commonly reported).
  • ✅ Adjusted R² → Compare multiple regression models with different numbers of features.
