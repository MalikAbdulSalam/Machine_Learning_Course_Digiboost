1. Machine Learning Overview

Machine Learning is a method where computers learn from data and improve automatically without being explicitly programmed.

Real Life Examples:

  • YouTube recommendations
  • Face recognition
  • Self-driving cars
  • Fraud detection

2. Data Preprocessing

Data preprocessing is the most important step in machine learning. It converts raw data into clean and usable format.

  • Remove missing values
  • Convert text into numbers
  • Fix wrong formats
  • Scale data

📌 Visual: Data Cleaning Workflow (see above)

3. Handling Missing Data

  • Remove empty rows
  • Replace with Mean
  • Replace with Median
  • Replace with Mode

Example: 10, 20, ?, 40 → Missing value replaced with 23 (Mean)

4. Converting Strings to Numbers

Machine learning models cannot understand text, so we convert it into numbers. 

Male   → 0
Female → 1
Red    → 0
Blue   → 1
Green  → 2

5. Feature Scaling

Scaling makes all values equal in range so model does not get biased.

  • Normalization (0 to 1)
  • Standardization (-1 to +1)

📌 Visual: Scaling Graph (see above)

6. Train-Test Split

  • Training Data → 80%
  • Testing Data → 20%

Training data teaches the model. Testing data checks performance.
