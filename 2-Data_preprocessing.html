Machine Learning & Data Preprocessing - Complete Lecture Notes

1. What is Machine Learning?

Machine Learning is a field of Artificial Intelligence (AI) where computers learn patterns from data and make predictions or decisions without being explicitly programmed.

How Machine Learning Works

  1. Collect Data
  2. Clean Data (Preprocessing)
  3. Train Model
  4. Test Model
  5. Use Model for Predictions

Real Life Examples

  • YouTube recommendations
  • Google Maps traffic prediction
  • Face unlock in mobile phones
  • Fraud detection in banks

2. What is Data Preprocessing?

Data preprocessing means converting raw data into clean and understandable format so machine learning models can use it.

Why preprocessing is important?

  • Real data is messy and incomplete
  • Machine learning models cannot handle text or missing values
  • Improves accuracy and performance

Example Problem

Dataset: 

Name   Age   Salary
Ali    25    50000
Sara   ?     60000
Ahmed  30    ?

This dataset cannot be directly used without cleaning.

3. Handling Missing Data

Missing data means some values are empty or unknown.

Types of Missing Data

  • Completely missing values
  • Incorrectly entered values

Methods to handle missing data

1. Remove Data

Remove rows or columns having missing values.

2. Mean Replacement

Replace missing value with average.

Example: 10, 20, 30 → Mean = 20

3. Median Replacement

Middle value is used (best for outliers).

4. Mode Replacement

Most repeated value used (for categories).

4. Converting Strings into Numbers

Machine learning models only understand numbers, not text. So we convert categorical data into numeric form.

Example: Gender Encoding

Male   → 0
Female → 1

Example: Color Encoding

Red   → 0
Blue  → 1
Green → 2

Why conversion is needed?

  • ML algorithms use mathematical calculations
  • Text cannot be processed directly

5. Handling Wrong Data Formats

Real datasets often contain incorrect formats.

Examples

  • 12,5 → 12.5 (decimal fix)
  • " Ali " → "Ali" (remove spaces)
  • Mixed case → standard format

Common Cleaning Steps

  1. Remove extra spaces
  2. Fix decimals
  3. Convert text to proper format
  4. Remove duplicate values

6. Feature Scaling (Normalization)

Feature scaling makes all values in dataset similar in range.

Why scaling is important?

  • Large values dominate small values
  • Improves training speed
  • Better accuracy

Methods

1. Min-Max Normalization

Scales data between 0 and 1.

X' = (X - Min) / (Max - Min)

2. Standardization (-1 to +1)

Used when data has different distributions.

X' = 2 * ((X - Min) / (Max - Min)) - 1

7. Splitting Dataset

Dataset is divided into training and testing parts.

Why split data?

  • To check model performance
  • To avoid overfitting

Common Split

Training Data = 80%
Testing Data  = 20%

Example

1000 samples:
800 → Training
200 → Testing

8. Complete Machine Learning Pipeline

  1. Collect Data from sources
  2. Clean Data (remove missing values)
  3. Convert text into numbers
  4. Fix wrong formats
  5. Scale data (-1 to +1)
  6. Split dataset (train/test)
  7. Train model
  8. Test model
  9. Deploy model
