📊 Regression · Classification · Clustering ML problem types

Three core approaches in Machine Learning — each solves a different kind of problem.
Feature Regression Classification Clustering
Purpose Predict a number Predict a category / class Group similar data without labels
Output Continuous value Discrete label Clusters (groups)
Learning Type Supervised Supervised Unsupervised
Example House price = $250,000 Email = Spam / Not Spam Group customers into 3 segments

📈 Regression Supervised

Used when the output is a number.
The model learns from labeled data.

  • Predict house price
  • Predict student marks
  • Predict tomorrow's temperature
Output:
House Price = $325,000

🏷️ Classification Supervised

Used when the output is a category (class).
The model learns from labeled data.

  • Spam or Not Spam
  • Disease or No Disease
  • Cat or Dog · Pass or Fail
Output:
Email = Spam

🔮 Clustering Unsupervised

Used when there are no labels.
The model finds groups of similar data automatically.

  • Customer segmentation
  • Group similar news articles
  • Organize photos by similar faces
Output:
Cluster 1 → Young Customers
Cluster 2 → Middle-aged
Cluster 3 → Senior

🏫 Simple Real-Life Example — Imagine a school

  • Regression → Predict a student's marks 85.5
  • Classification → Predict whether the student will Pass or Fail
  • Clustering → Automatically group students into High Performers, Average, and Weak based on their data
🧠 Easy Way to Remember 📊 Regression = Predict a Number (Price, Salary, Temperature) 🏷️ Classification = Predict a Category (Yes/No, Cat/Dog, Spam) 🔮 Clustering = Find Similar Groups (No labels required)
✅ Regression → Numerical prediction ✅ Classification → Category prediction ✅ Clustering → Automatic grouping of similar data
⚡ Supervised: uses labeled data ⚡ Unsupervised: finds hidden patterns 🧩 scikit-learn · TensorFlow · PyTorch
