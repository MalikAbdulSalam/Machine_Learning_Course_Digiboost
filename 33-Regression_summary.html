📈 Linear · Multiple · Polynomial Regression types

The main difference is the number of input features and the type of relationship the model learns.
Feature Linear Regression Multiple Linear Regression Polynomial Regression
Number of Input Features 1 2 or more 1 or more
Relationship Linear Linear Non-linear (curved)
Equation y = mx + c y = b₀ + b₁x₁ + b₂x₂ + … + bₙxₙ y = b₀ + b₁x + b₂x² + b₃x³ + …
Best For One variable affects the target Multiple variables affect the target Curved relationships

1. Linear Regression 1 feature · straight line

Uses one independent variable to predict one dependent variable.

y = mx + c   (ŷ = b₀ + b₁x)
🏠 Predict House Price using House Size only.
Example dataset:
House Size (sq ft) → Price ($)
1000 → 150000  ·  1200 → 180000  ·  1500 → 220000
📌 Fit: ŷ = 8.49 - 0.54x  ·  R² = 0.72

2. Multiple Linear Regression 2+ features · straight

Uses two or more independent variables to predict one dependent variable.

y = b₀ + b₁x₁ + b₂x₂ + … + bₙxₙ
🏠 Predict House Price using: Size + Bedrooms + Age
Example dataset:
Size   Bedrooms   Age   →   Price
1000   2   10   →   150000
1200   3   8   →   180000
1500   4   5   →   220000

3. Polynomial Regression curved relationship

Uses polynomial terms (x², x³, …) to model a curved relationship.

y = b₀ + b₁x + b₂x² + b₃x³ + …
🌾 Predict crop yield from fertilizer amount (yield increases then decreases)
Example dataset:
Fertilizer (kg) → Yield (tons)
10 → 20  ·  20 → 38  ·  30 → 52  ·  40 → 55  ·  50 → 50
📌 A straight line cannot capture this pattern — a curve fits perfectly.
📌 Rule to remember: 1 feature + straight line → Linear Regression Multiple features + straight line → Multiple Linear Regression Curved relationship → Polynomial Regression

🧪 Simple code for each (scikit-learn)

1. Linear Regression (one feature)

import pandas as pd from sklearn.linear_model import LinearRegression # Data data = { "Size": [1000, 1200, 1500, 1800], "Price": [150000, 180000, 220000, 260000] } df = pd.DataFrame(data) X = df[["Size"]] y = df["Price"] model = LinearRegression() model.fit(X, y) prediction = model.predict([[1600]]) print("Predicted Price:", prediction[0])

2. Multiple Linear Regression (multiple features)

import pandas as pd from sklearn.linear_model import LinearRegression # Data data = { "Size": [1000, 1200, 1500, 1800], "Bedrooms": [2, 3, 3, 4], "Age": [10, 8, 5, 2], "Price": [150000, 180000, 220000, 260000] } df = pd.DataFrame(data) X = df[["Size", "Bedrooms", "Age"]] y = df["Price"] model = LinearRegression() model.fit(X, y) prediction = model.predict([[1600, 3, 4]]) print("Predicted Price:", prediction[0])

3. Polynomial Regression (curved)

import pandas as pd from sklearn.preprocessing import PolynomialFeatures from sklearn.linear_model import LinearRegression # Data data = { "Fertilizer": [10, 20, 30, 40, 50], "Yield": [20, 38, 52, 55, 50] } df = pd.DataFrame(data) X = df[["Fertilizer"]] y = df["Yield"] poly = PolynomialFeatures(degree=2) X_poly = poly.fit_transform(X) model = LinearRegression() model.fit(X_poly, y) prediction = model.predict(poly.transform([[35]])) print("Predicted Yield:", prediction[0])

🔑 Key Difference in Code

Linear
X = df[["Size"]]
Multiple Linear
X = df[["Size", "Bedrooms", "Age"]]
Polynomial
poly = PolynomialFeatures(degree=2)
X_poly = poly.fit_transform(X)
💡 Key insight
Polynomial regression converts features into x, x², x³, … before fitting a linear model.
📌 Linear Regression: One input feature, straight-line relationship. 📌 Multiple Linear Regression: Multiple input features, still a straight-line relationship in feature space. 📌 Polynomial Regression: One or more input features transformed with polynomial terms to model curved relationships.
